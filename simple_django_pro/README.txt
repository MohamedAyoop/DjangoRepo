Procedure to execute the project

# Step 1: Create virtual environment
python -m venv myenv

# Step 2: Activate virtual environment
# On Windows:
myenv\Scripts\activate


# Step 3: Create your project folder
mkdir myproject
cd myproject

# Step 4: Extract the folder into python_project folder
then navigate to the project
cd myproject\simple_django_porject

# Step 5:create a database and add the specific criterias in settings.py

# Step 6:manage.py makemigrations

# Step 7:manage.py migrate

# Step 8:manage.py runserver
